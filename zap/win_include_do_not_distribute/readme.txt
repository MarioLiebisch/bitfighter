This folder contains files you will need to compile Bitfighter on the Windows platform.  The following files are required, but not distributed with the source for copyright reasons:

dinput.h
dsound.h
afxres.h
winres.h

These files can be obtained by downloading the MS DirectX SDK and/or Platform SDK, as per the following:

http://www.gamedev.net/community/forums/topic.asp?topic_id=441623

Sorry for the inconvenience.
